# P0 SOP 到 GCFIS 单据映射表

日期：2026-05-25  
状态：目标模式产出  
来源：`/Users/lujunxiang/Desktop/我的工作/工业绿链/项目/葛化/方案与规划/葛化工厂作业标准化规范及sop体系建设指南.md`

## 1. 映射原则

1. GFIS 运行层是工厂运行主体，所有 SOP 必须基于 GFIS 运行层实现。
2. SOP 实现和验收必须落在 GFIS 运行层的标准单据、`gcfis_custom` 自定义对象、工作流、角色权限、报表、附件和运行态 API 上。
3. GFIS Demo 只用于展示、培训、追溯说明和前端回归，不作为 SOP 实现主体，不作为业务验收主体，也不能替代运行层单据事实。
4. GCFIS 优先复用底层标准单据能力。
5. 自定义字段优先于自定义 DocType。
6. 工作流配置优先于代码二开。
7. 只有涉及葛化专用批次、绿色供应链、平台同步、SOP绑定和供应链金融凭证链时，才进入 `gcfis_custom`。
8. 所有 P0 SOP 必须能在 GFIS 运行层中留下单据、状态、责任人、时间和证据。

## 3. 台账映射

| 台账 | 底层标准落点 | GCFIS 处理 |
|---|---|---|
| SOP总台账 | 自定义单据 | `gcfis_custom` 建 `GCFIS SOP台账` |
| 订单台账 | 销售订单 | 标准列表+自定义报表 |
| 报价台账 | 报价单 | 标准列表+报价版本字段 |
| 打样台账 | 项目、任务、物料、物料清单 | MVP 用配置，二期建 `GCFIS打样申请` |
| 供应商台账 | 供应商 | 标准能力 |
| 原料批次台账 | 批次 + 采购收货 + 质量检验 | 标准能力+字段 |
| 生产计划台账 | 生产计划 + 生产工单 | 标准能力 |
| 生产日报台账 | 作业卡 + 生产工单 + 库存出入库 | 自定义报表 |
| 质量检验台账 | 质量检验 | 标准能力 |
| 不合格品台账 | 质量检验 + 库存出入库 + 仓库 | 标准能力+报表 |
| 成品库存台账 | 库存账本、仓库、批次 | 标准能力 |
| 发货回单台账 | 发货单、发运单 | 需 POD 字段或单据 |
| 设备台账 | 资产 | 标准能力 |
| 安全隐患台账 | 问题单、项目任务 | MVP 配置，后续 `GCFIS安全隐患` |
| 应收账款台账 | 销售发票 + 收付款凭证 | 验证状态，不做正式财务上线 |
| 风险问题台账 | 问题单 | 标准能力+类型字段 |
| 决策事项台账 | 自定义单据/项目任务 | 可与飞书/Bitable 保持主权分离 |

## 4. 首轮覆盖判断

| 分类 | 数量 | 判断 |
|---|---:|---|
| 标准能力可覆盖 | 8 | 报价单、销售订单、采购收货、库存出入库、生产工单、作业卡、质量检验、资产 |
| 配置/字段可覆盖 | 5 | 订单评审、回收料、排产建议、发货回单、设备点检 |
| 建议二开 | 2 | 样品全流程、工艺参数/安全隐患深度模板 |

首轮估计覆盖率：`13/15 = 86.7%` 可通过标准能力+配置覆盖，满足 GCFIS 原型验证门槛。
