# GFIS 现场调研数据采集表

Round ID：`GPCF-GF-LR-001`

## 采集原则

- 只采集能支撑 GFIS 工厂事实主账的数据。
- 每项数据必须能对应岗位、单据、系统字段或门禁。
- 培训资料只能作为准备输入，不能替代现场真实记录。

## 采集表

| 主题 | 采集项 | 样例/口径 | 用途 | 优先级 |
|---|---|---|---|---|
| 工单 | 工单编号规则 | 年月日+产线+序号 | 工单主键 | P0 |
| 工单 | 工单状态 | 待排产/生产中/暂停/完工/关闭 | 状态机 | P0 |
| 配料 | 原料批次规则 | 供应商+日期+批号 | 原料追溯 | P0 |
| 配料 | 配方版本 | 产品+版本+生效日期 | 工艺控制 | P0 |
| 生产 | 产线编号 | 线别、设备组 | 产能和排产 | P0 |
| 生产 | 班组排班 | 班次、人员、岗位 | 报工责任 | P1 |
| 生产 | 单线产能 | 件/小时或箱/班 | 计划与绩效 | P1 |
| 质量 | 首件确认项 | 尺寸、外观、承重 | 量产门禁 | P0 |
| 质量 | 巡检频率 | 每小时/每批次 | 过程控制 | P0 |
| 质量 | 不合格处理 | 隔离/返工/报废 | WAES 门禁 | P0 |
| 库存 | 库位编码 | 区-排-层-位 | 库存准确性 | P0 |
| 库存 | 入库规则 | 检验合格后入库 | 质量入库门禁 | P0 |
| 出库 | 出库复核项 | 批次、数量、目的地 | 发货事实 | P0 |
| 设备 | 点检项 | 温度、压力、电流、润滑 | 安全和维修 | P1 |
| 设备 | 故障等级 | 一般/停线/安全 | 异常升级 | P1 |
| 追溯 | 追溯码规则 | 工单+批次+序号 | 全链路追溯 | P0 |
| 指标 | 成品率 | 合格数/生产数 | 质量绩效 | P1 |
| 指标 | OEE | 可用率*性能*质量 | 设备绩效 | P2 |
| 指标 | 单耗 | 原料/电/人工 | 成本事实 | P2 |
