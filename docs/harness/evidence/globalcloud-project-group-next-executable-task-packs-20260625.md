## 5. 停止条件

任一任务触发以下条件时，本任务包不得继续升级状态：

- 需要真实业务 owner、scenario owner、客户、外部系统或生产环境确认但未获得确认；
- 需要 stage、commit、push、delete、deploy、release、merge、生产写入或权限变更；
- 任一项目命令失败且没有 repair evidence；
- 证据与项目状态或依赖矩阵冲突；
- 出现 TOKEN、密钥、敏感路径或未授权外部写入。
