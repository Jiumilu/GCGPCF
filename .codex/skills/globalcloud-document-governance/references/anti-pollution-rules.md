# Anti-Pollution Rules

## 禁止旧口径

- `GPC-Native`
- `gpc-native`
- `GPC Native`
- `GPC\-Native`
- 英文独立词 `Native`
- `协同中台`

## 禁止旧 AI 定位

- `提示词工程服务`
- `数字意识框架`
- `Agent框架`
- `XiaoC 管 prompt`
- `XiaoC 提供提示词`
- `桌面/语音/社交 AI 入口`
- `XiaoC + Hermes + XGD`
- `XiaoC、Hermes、XGD`

## 禁止真实性污染

- 把设计完成写成实现完成。
- 把文档通过写成业务完成。
- 把控制塔可见写成业务闭环完成。
- 把 KDS 本地镜像写成真实 KDS API 已同步。
- 把 Brain 写成 KDS 主存替代。
- 把 WAES 写成业务主账。
- 把 GPC 写成 GFIS 工厂事实主账。
