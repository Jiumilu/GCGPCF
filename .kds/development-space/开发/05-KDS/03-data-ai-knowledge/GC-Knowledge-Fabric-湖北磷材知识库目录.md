# GC-Knowledge Fabric 湖北磷材知识库目录

## 1. 定位

湖北磷材知识库是 P2 阶段并行线，优先建设拓厂项目、原料/行业/订单和新工厂复制模板知识体系。该目录不依赖 GFIS 深度实施即可先行组织知识对象。

## 2. 拓厂项目知识库

| 子目录 | 默认挂池 | 说明 |
|---|---|---|
| 新工厂选址 | 场景池 + 政策池 | 区域、土地、政策、协同条件 |
| 建设条件 | 场景池 + 产能池 + 装备池 | 建设要素、配套条件 |
| 设备需求 | 装备池 + 产能池 | 设备清单、产线规划 |
| 合作单位资料 | 人才池 + 场景池 | 参与方、服务商、责任边界 |
| 建设期风险 | 数据池 + 场景池 | 缺口、风险、审批事项 |
| 预运营准备 | 订单池 + 产能池 + 运力池 | 预运营订单、承接责任 |

## 3. 原料 / 行业 / 订单知识库

| 子目录 | 默认挂池 | 说明 |
|---|---|---|
| 磷材原料 | 原料池 | 原料来源、质量、供应商 |
| 市场供需 | 原料池 + 场景池 | 供需、价格、上下游变化 |
| 行业价格 | 原料池 + 数据池 | 价格记录与来源 |
| 上下游客户 | 订单池 + 原料池 | 客户、供应商、渠道 |
| 销售订单 | 订单池 + 资金池 | 销售机会与订单事实 |
| 采购订单 | 原料池 + 资金池 | 采购机会与订单事实 |
| 政策影响 | 政策池 + 原料池 | 政策对原料和订单的影响 |

## 4. 新工厂复制模板

新工厂复制模板只作为候选结构，不自动形成业务事实。每个新工厂必须重新登记来源、证据、责任边界、WAES 门禁和确认记录。

## 5. P2 输出边界

- 可形成拓厂知识包。
- 可形成原料、行业、订单问答目录。
- 可记录潜在产值与渠道贡献。
- 不自动进入正式收益分配。
- 不自动视为 GFIS 正式写回完成。
