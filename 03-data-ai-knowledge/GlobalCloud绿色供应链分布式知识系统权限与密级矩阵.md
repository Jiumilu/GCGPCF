# GlobalCloud 绿色供应链分布式知识系统权限与密级矩阵

日期：2026-06-20
轮次：`GPCF-KDS-DKS-059`
状态：`controlled_permission_classification_matrix`

## 1. 状态边界

本文定义绿色供应链分布式知识系统 P0 阶段的 DSR 密级、角色权限、RAG 准入、安全调用和跨单位可见边界。本文不授予真实系统权限，不创建真实账号，不开放敏感资料，不替代人工授权或委员会裁决。

## 2. DSR 密级

| 密级 | 定义 | 示例 | 默认 RAG |
|---|---|---|---|
| `DSR-L0` | 公开资料 | 公开政策、公开标准、公开行业信息 | limited / safe_after_gate |
| `DSR-L1` | 组织内部一般资料 | SOP 模板、公开化项目目录、一般培训资料 | limited |
| `DSR-L2` | 项目或合作单位受限资料 | 葛化建设/运营资料、湖北磷材资料包、订单线索、OEM 摘要 | repair_required / limited_after_gate |
| `DSR-L3` | 敏感或封存资料 | 金融凭证、账户、金额明细、合同全文、POD 原件、质量争议、客户敏感资料 | sensitive_metadata_only / blocked |

DSR-L3 默认只允许记录编号、摘要、哈希、保管人、密级、原件封存位置和授权状态，不进入开放问答正文。

## 3. 角色矩阵

| 角色 | 可见范围 | 可执行动作 | 禁止动作 |
|---|---|---|---|
| 用户治理代表 | 全局治理视图 | 规则治理、急停、授权边界确认、备案 | 代替委员会日常裁决 |
| KDS 记录人 | 授权项目知识对象 | 登记来源、证据、对象、挂池、状态 | 确认业务事实、分配收益 |
| WAES 治理人 | 门禁输入和必要证据 | 记录规则、阻断、人工确认路径、委员会路径 | 写业务主账、替代业务验收 |
| GFIS 项目负责人 | 葛化授权资料和 GFIS 候选对象 | 人工核验、评测签认、候选写回确认 | 绕过 WAES 或缺证写主账 |
| 湖北磷材负责人 | 本单位资料和受邀事项 | 资料提交、拓厂评估确认、缺口补证 | 查看他方未授权资料 |
| 合作单位代表 | 本单位资料、本单位贡献、受邀悬赏 | 提交资料、查看本单位积分/额度/缺口 | 查看他方明细、确认收益分配 |
| 财务联系人 | 授权财务索引和统计口径 | 开票/到账过程口径复核、凭证保管确认 | 向无关角色公开敏感凭证 |
| 委员会成员 | 待裁决事项和必要证据 | 多数决、冻结、扣减、分配、争议处理 | 扩大访问范围、绕过备案 |
| 外部专家 | 被邀请资料 | 专项评审、政策/标准/行业判断 | 访问未邀请资料或 DSR-L3 原件 |
| AI 助手 | 已准入知识和当前会话授权上下文 | 问答、候选事实、候选 SOP、缺口、风险、字段建议 | 确认事实、写主账、结算、裁决 |

## 4. 操作权限

| 操作 | DSR-L0 | DSR-L1 | DSR-L2 | DSR-L3 |
|---|---|---|---|---|
| 知识问答 | 允许 | 允许 | 授权内 limited | 默认禁止，仅元数据 |
| RAG 强引用 | gate 后允许 | gate 后有限允许 | 人工确认后有限允许 | 禁止，除非脱敏摘要另行批准 |
| AI 候选事实生成 | 允许 | 允许 | 授权内允许 | 仅基于脱敏索引 |
| SOP 建议 | 允许 | 允许 | 授权内允许 | 仅输出流程建议，不引用原文 |
| 跨单位共享 | 允许 | 需授权 | 需双方确认 | 禁止或委员会确认 |
| 导出 | 允许 | 需记录 | 需授权和水印 | 默认禁止 |
| 收益/积分引用 | 可引用 | 可引用 | 需人工确认 | 需委员会或财务确认 |
| 写回候选 | 可候选 | 可候选 | WAES + 人工 | 默认 blocked |

## 5. 资料类型默认密级

| 资料 | 默认密级 | 处理方式 |
|---|---|---|
| 权威政策 / 标准网站 | DSR-L0 / DSR-L1 | 保留来源、检索时间、适用范围 |
| 葛化建设资料 | DSR-L1 / DSR-L2 | 项目授权内使用 |
| 葛化工厂运营资料 | DSR-L2 | 项目授权内问答和验收 |
| 葛化订单资料 | DSR-L2 / DSR-L3 | 脱敏索引优先 |
| 辽宁远航客户确认 / 报价 / 合同链 | DSR-L2 / DSR-L3 | 原件封存，KDS 记录索引 |
| 现代精工 OEM 资料 | DSR-L2 / DSR-L3 | 责任拆分摘要可用，原件受控 |
| 质量资料 | DSR-L2 / DSR-L3 | 质量结论需人工确认 |
| 发货 / POD | DSR-L2 / DSR-L3 | POD 原件默认封存 |
| 金融凭证 | DSR-L3 | 仅脱敏索引，不开放原文 |
| 湖北磷材原料 / 行业资料 | DSR-L1 / DSR-L2 / DSR-L3 | 按供应商、价格、订单敏感性分级 |
| AI 输出 | 继承来源最高密级 | 默认候选，不降低密级 |

## 6. RAG 准入规则

| 条件 | 准入 |
|---|---|
| T0/T1 且 DSR-L0/L1 且来源完整 | `safe` 或 `limited` |
| T2 且 DSR-L1/L2 且人工验收通过 | `limited`，必要时 `safe` |
| T3/T4 或证据不全 | `repair_required` |
| T5 LLM 分析 | `blocked` |
| DSR-L3 原件 | `sensitive_metadata_only` 或 `blocked` |
| committee_review_required | `blocked` 至裁决完成 |
| waes_blocked | `blocked` |

## 7. 跨单位权限

| 场景 | 规则 |
|---|---|
| 合作单位查看自己的贡献 | 允许 |
| 合作单位查看自己的 AI 额度 | 允许 |
| 合作单位查看他方资料 | 默认禁止，除非被邀请或参与悬赏 |
| 葛化母版复用到湖北磷材 | 只复用结构、字段、SOP 模板，不复用未确认事实 |
| 湖北磷材资料反哺葛化 | 需双方确认或悬赏/委员会路径 |
| 第三方专家评审 | 只看被邀请资料，必须备案 |

## 8. 人工和委员会门禁

必须人工确认：

1. 资料密级和可见范围。
2. DSR-L2 资料进入 AI/RAG。
3. DSR-L3 资料脱敏索引和封存位置。
4. GFIS 候选写回进入人工录入。
5. 合作单位资料跨单位共享。

必须委员会确认：

1. 重大违规、泄密、伪造、越权确认。
2. 收益分配和争议。
3. 积分重大争议或追溯扣减。
4. 悬赏结算争议。
5. 潜在产值转正式产值。
6. 跨单位权益争议。

## 9. 禁止事项

1. 不得用 AI 输出降低资料密级。
2. 不得把 DSR-L3 原件进入普通 RAG。
3. 不得把合作单位自购额度暴露给无关单位。
4. 不得把候选收益、候选积分、候选悬赏对外显示为已确认。
5. 不得把 WAES governance_recorded 写成业务批准。
6. 不得把 KDS 本地镜像写成真实 KDS API 已同步。
